# RainlySSH
一款基于 Golang 开发的超轻量化 WebSSH 工具，通过将 HTTP 升级为 Websocket 构建 SSH 客户端，配合 xterm 实现浏览器端的在线 SSH 管理，同时内置系统监控能力，兼顾易用性与轻量化。
<br><br>
# ✨ 核心特性

> 🚀 跨平台运行
*   **完美支持 Windows 和 Linux 系统，无需额外依赖，下载对应平台可执行文件。**

> 🌐 WebSSH 核心能力<br><br>
*   **HTTP 协议升级为 Websocket 协议，保障 SSH 通信的实时性与稳定性**
*   **集成 xterm 实现浏览器端原生 SSH 终端交互，操作体验接近本地终端**
*   **支持 SSH 命令双向交换，执行结果实时回显**
  
> 📊 系统监控<br><br>
*   **自动采集服务器核心运行数据，无需额外配置**
*   **进程列表（PID、名称、占用资源等）**
*   **CPU 使用率、内存占用率**
*   **网络带宽实时占用信息**

> 🔧 极简部署<br><br>
*   **单文件运行，无需复杂配置，一分钟完成部署，开箱即用。**

# 📖 使用说明<br><br>
*   **浏览器访问 http://[服务器IP]:8667（默认端口，可自定义），输入 SSH 连接信息（主机、端口、用户名、密码），即可进入 WebSSH 终端**
*   **执行任意 SSH 命令，结果实时返回 / 系统监控数据（进程、CPU、内存、带宽）**
